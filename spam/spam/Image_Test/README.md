Working with the Camera: Tips&Tricks


Install Zbar
------------------
sudo apt-get install libzbar-dev
sudo -H pip3 install setuptools --upgrade
sudo -H pip3 install zbarlight


Install OpenCV
-----------------------
sudo pip3 install opencv-python
